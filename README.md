## Ключевые проекты
### Агрегатор игрового контента с гибридной рекомендательной системой
Построил масштабную платформу-агрегатор видеоигр, объединяющую функции социальной сети, рекомендательного сервиса и медиаресурса. Основной инновацией стала гибридная рекомендательная система, сочетающая методы коллаборативной фильтрации и контентного анализа с элементами машинного обучения на Python. Система анализирует поведение пользователей, сравнивает игровые атрибуты и формирует персонализированные подборки. Реализовал механизм агрегации данных из 10+ внешних источников (Steam, IGDB, Metacritic) с последующей нормализацией и унификацией. Для сообщества создал систему геймификации с уровнями, достижениями и возможностью публикации экспертных материалов с модерацией и рейтингованием.

### AI-конструктор интерьеров с генеративной заменой товаров
Разработал инновационную платформу для дизайна интерьеров, где пользователи могут генерировать реалистичные визуализации комнат с товарами из каталога. Ключевая особенность — интеллектуальный пайплайн, начинающийся с заполнения брифа (фото помещения, референсы, стиль, бюджет), на основе которого LLM-модель подбирает подходящие товары из базы. Затем система генерирует изображение интерьера с этими товарами, после чего пользователь может заменять любой элемент на аналогичный из каталога через интуитивный интерфейс. Реализовал интеграцию с генеративными AI-моделями, систему семантического поиска товаров и механизм «умной» замены с сохранением стилистического единства сцены.

### Киберспортивная платформа для мобильных игр
Спроектировал и реализовал комплексную экосистему для киберспортивных дисциплин на мобильных устройствах, включающую все компоненты профессиональной игровой платформы. Разработал систему рейтинговых матчей с автоматическим подбором соперников, визуализацией уровней и статистикой игроков. Для верификации результатов внедрил OCR-модуль, анализирующий скриншоты финальных экранов с точностью распознавания 98.5%. Создал внутреннюю экономику с виртуальной валютой, магазином скинов и системой наград. Реализовал админ-панель для управления турнирами, разрешения спорных ситуаций и модерации контента. Платформа поддерживает одновременную работу 10 000+ активных пользователей с минимальной задержкой при формировании лобби.

### Комплексная экосистема интернет-магазина сантехники
Разработал и внедрил полноценную платформу для онлайн-торговли сантехническим оборудованием, включающую каталог из 15 000+ товарных позиций. Проект представлял собой решение «под ключ»: от создания дизайна и фирменного стиля до автоматизации бизнес-процессов закупки и продаж. Особое внимание было уделено автоматизации работы с товарными данными — реализовал интеллектуальные скрипты на Python для парсинга информации из различных источников, обработки изображений с использованием AI-алгоритмов и автоматического заполнения базы данных. Для управления контентом создал интуитивную административную панель с пакетным редактированием и системой импорта. В результате клиент получил полностью готовый к работе магазин с настроенными интеграциями платежных систем и служб доставки, что позволило запустить бизнес в кратчайшие сроки.

### Дизайн-платформа для мира крафтового пива
Совместно с заказчиком, экспертом в пивоваренной индустрии, создал цифровую платформу, сочетающую образовательный контент и эстетику премиального бренда. Проект стал результатом глубокого партнерства — каждый элемент интерфейса, от интерактивной карты мировых пивоварен до детальных сенсорных описаний сортов, разрабатывался в ходе совместных воркшопов и мозговых штурмов. Основной задачей было преодоление ограничений на онлайн-продажу алкоголя в РФ через смещение фокуса с коммерции на сторителлинг и формирование сообщества ценителей. Реализовал систему динамического контента, адаптивную верстку с акцентом на кинематографичную визуализацию и удобные инструменты для публикации статей и обзоров.
